Problem: Find the time complexity for a nested for loop. 

{% highlight java linenos %}
A() {
    int i, j, k ,n;
    for(i = 1; i <= n; i++) {
        for(j = 1; j <= i^2; j++) {
            for(k = 1; k < n/2; k++) {
                System.out.println("hello world"); 
            }
        }
    }
}
{% endhighlight %}



Answer: O(n^4)
